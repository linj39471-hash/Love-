## Stockfish.js

This extension can load Stockfish.js 18.0.8, lite single-threaded WASM build.

- Project: Stockfish.js

- Source: https://github.com/nmrugg/stockfish.js
- License: GNU General Public License version 3

Stockfish.js is based on the Stockfish chess engine:

- Source: https://github.com/official-stockfish/Stockfish
- License: GNU General Public License version 3

The extension does not alter the Stockfish engine files. When local engine files are not present, it downloads the published npm build from UNPKG at runtime.
